Parameters

context

Context that is used to get icon drawable. It isn't stored in class so it won't call memory leak

swipeDirs

Binary OR of next directions: ItemTouchHelper.LEFT, ItemTouchHelper.RIGHT, ItemTouchHelper.START and ItemTouchHelper.END, for more info see ItemTouchHelper.SimpleCallback

onSwipedListener

Function that will be called when item were swiped in one of supported directions

color

Color of background that will be drawn when user swipes item

icon

DrawableRes of icon that will be displayed when user swipes item. If you don't want to draw any icon pass 0 (it is the default value)
