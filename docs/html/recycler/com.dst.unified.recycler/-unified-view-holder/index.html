UnifiedViewHolder

abstract class UnifiedViewHolder<T : RecyclerItem>(itemView: View) : RecyclerView.ViewHolder
Content copied to clipboard

This is a class that is responsible for displaying classes that implement RecyclerItem in RecyclerView.

All your UI logic for displaying a class should go here. In init block you can set references to all your views using itemView.findViewById and in bind method display appropriate data in them

For example:

class MyItemHolder : UnifiedViewHolder<MyItem> {
    val myView: MyView

    init {
        myView = itemView.findViewById(R.id.myView)
    }

    override fun bind(item: MyItem, position: Int) {
        myView.setMyData(item.myData)
    }
}

Parameters

T

Type of the item that is going to be displayed in that ViewHolder. That type has to implement RecyclerItem and return same RecyclerItem.viewType as appropriate UnifiedViewHolderType.viewType for that UnifiedViewHolder

itemView

Root View for that ViewHolder. For more info see RecyclerView.ViewHolder.itemView
